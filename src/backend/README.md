## Getting Started

### Prerequisites

- JDK 17+ (tested with JDK 24)

### Running the Backend

```bash
# PowerShell or CMD
.\gradlew.bat bootRun

# Bash
./gradlew bootRun
```

The API will be available at http://localhost:8080.

**NOTES**:
- Ensure the `JAVA_HOME` environment variable is set properly.
- Ensure the Java `bin` directory is in the `PATH` environment variable.

### Running Tests

```bash
# PowerShell or CMD
.\gradlew.bat test

# Bash
./gradlew test
```
